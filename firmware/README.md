# Firmware

Arduino sketches for the balance robot live here.

## Canonical sketches

- `sbraman/balancebot_main.ino`: main balance and actuation firmware.
- `simplepid/simplepid.ino`: simpler PID tuning sketch set.
- `task5/bb_1545_task5.ino`: task-specific integration sketch.

The remaining sketches in these folders are preserved as legacy tuning variants.
