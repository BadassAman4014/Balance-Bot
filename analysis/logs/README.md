# Logs

This folder contains consolidated run outputs and tuning logs moved out of the task folders.

## Naming

Files use a source-based pattern so repeated runs stay unique. For example:

- `simulation_task2a_task2a_output24.txt`
- `simulation_task1b_Task1B_windows_task1b_output.txt`

Add new logs here instead of leaving them beside the controllers or firmware sketches.
